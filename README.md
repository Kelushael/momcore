## рҹ’» Installation (Windows)

### The Easy Way (recommended)

1. **Download** this repo (green "Code" button вҶ’ Download ZIP)
2. **Unzip** it to your Desktop or Documents
3. **Double-click** `install.bat`
4. Follow the prompts вҖ” it installs everything you need
5. When it's done, double-click **M.O.M.-CORE** on your Desktop!

That's it. The installer handles Node.js, Ollama, and the AI model.

### Starting It Later

Just double-click the **M.O.M.-CORE** shortcut on your Desktop.

Or double-click `start.bat` in this folder.

### For Developers

```bash

ollama serve      # start local AI (if not running)
npx vite          # start dev server вҶ’ http://localhost:5173
```
